Q: What is OOP?
Answer:

OOP (Object Oriented Programming) ek programming paradigm hai jo real world entities ko objects aur classes ki form me represent karta hai.
Iska main purpose code ko modular, reusable aur maintainable banana hai.
OOP 4 main pillars pe based hoti hai: Encapsulation, Inheritance, Polymorphism aur Abstraction.

------------------------------------------------------------------------------------------------------------------------------------------------------------

Q: Difference between class and object?

Answer:

Class ek blueprint hoti hai.

Object us blueprint ka real instance hota hai.


------------------------------------------------------------------------------------------------------------------------------------------------------------

Q: What is constructor?

Answer:

Constructor ek special method hota hai jo object create hone par automatically execute hota hai.
Iska use object ki initial values set karne ke liye hota hai.

------------------------------------------------------------------------------------------------------------------------------------------------------------

Q: What is inheritance?

Answer:

Inheritance ek mechanism hai jisme ek class dusri class ki properties aur methods ko inherit karti hai.

------------------------------------------------------------------------------------------------------------------------------------------------------------

Q: What is encapsulation?

Answer:

Encapsulation ka matlab hai:

Data aur methods ko ek single unit (class) me wrap kar dena.

------------------------------------------------------------------------------------------------------------------------------------------------------------

Q: What is abstraction?

Answer:

Abstraction ka matlab hai unnecessary details hide karna aur sirf required functionality show karna.

------------------------------------------------------------------------------------------------------------------------------------------------------------

Q: What is polymorphism?

Answer (Strong Answer):

Polymorphism ka matlab hai: Same method name but different implementation.
